#Home Network Security Lab 

##Objective
Analyxe,identify and mitigate vulnerabilities in a local home network

#Environment

-Main system:Kali Linux
-Network: Home network
-Devices analyzed:
    -TV BOX
    -Router
    -Mobile devices
-Tools
    -nmap
    -Router configuration interface
    -MiniUPnPc(upnpc)

-Methodology
    -Network scanning
    -Service enumeration
    -Nmap output analysis
    -Identification of potential attack vectors

##Disclaimer

This project is conducted strictly in a controlled environment(,y own network)for educational purposes only.
Not unauthorized systems or networks are targeted.
